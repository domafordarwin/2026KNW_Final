# 사용자 매뉴얼 (User Manual)

## 1. 관리자(Admin)
### 1.1 문항 등록
1) 지문 등록 → 2) 문항 등록(영역/하위지표/난이도/유형) → 3) 정답/루브릭 설정 → 4) 검사 버전 발행

### 1.2 추천 도서 Allowlist 등록
- 도서 CRUD 또는 CSV 업로드
- 업로드 후 검증 결과 확인(필수 컬럼/중복/비활성)
- 정책: “목록 외 추천 금지” 유지 권장

## 2. 학교 담당자(SchoolManager)
### 2.1 학생 계정 관리
- CSV 업로드(학번/반/이름 등) → 계정 발급 → 초기 비밀번호 배포/초기화
### 2.2 학교 보고서 출력
- 세션 선택 → 학교 보고서 생성 → 열람/다운로드

## 3. 검사 진행 교사(Teacher)
### 3.1 검사 배부
- 세션 생성(학급/버전/기간/정책) → 코드/링크/QR 배포 → 진행 현황 확인
### 3.2 학생 기록 관리
- 미제출 확인, 재응시 승인, 제출 잠금/해제
### 3.3 AI 피드백 검수/첨삭
- 학생 선택 → AI 초안 확인 → 수정/코멘트 → 섹션 재생성(필요 시) → 승인(Approve)
### 3.4 최종 보고서
- 승인 완료 후 최종 보고서 생성 → 학생/학부모 공유 링크 발급(만료 설정)

## 4. 학생(Student)
- 로그인 → 검사 응시/제출 → 보고서 열람/다운로드(허용 시)

## 5. 학부모(Parent)
- 공유 링크 접속 또는 계정 연동 → 자녀 보고서 열람/다운로드
- 링크 만료 시 교사에게 재발급 요청
